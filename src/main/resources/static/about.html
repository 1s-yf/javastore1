项目简介

电脑商城 是一个基于 Spring Boot + MyBatis + MySQL 构建的 B2C 在线购物平台，项目从零开始，完整实现了电商系统的核心功能。 我们致力于为在校学生和年轻群体提供高品质、高性价比的电脑外设、整机及数码产品，同时为 Java 开发者提供一个优秀的学习案例。

项目采用前后端分离架构（前端静态页面 + 后端 RESTful API），涵盖用户管理、商品浏览、购物车、订单处理等典型业务场景。 通过本项目，您可以深入了解 Spring Boot 整合 MyBatis、登录拦截器、AOP 统一日志、文件上传、异常处理等企业级开发技术。

目前商城已上线运营（演示版），我们将持续迭代，加入更多前沿技术，如 Redis 缓存、Elasticsearch 搜索、Spring Security 权限控制等， 打造一个稳定、高效、可扩展的电商平台。

核心功能

  • 用户注册 / 登录（MD5 加密）
  • 个人资料修改 & 头像上传
  • 收货地址管理（增删改查 + 默认地址）
  • 省市区三级联动（3523 条行政区划）
  • 商品热销排行（首页推荐）
  • 购物车（加入/删除/数量修改）
  • 订单生成 & 支付模拟
  • 订单列表 & 订单详情

注：购物车及订单模块目前处于快速迭代中，更多功能即将上线。
