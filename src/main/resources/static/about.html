欢迎访问电脑商城

项目简介

电脑商城 是合肥工业大学宣城校区2023级信息安全专业《软件工程》课程的实训项目。 项目基于 Spring Boot + MyBatis + MySQL 构建，是一个完整的 B2C 在线购物平台。

本项目采用前后端分离架构，涵盖需求分析、概要设计、详细设计、编码实现、测试维护等软件工程全生命周期。 通过本项目的开发，我们深入实践了软件工程方法论、团队协作流程以及企业级开发技术栈。

技术架构

后端框架

Spring Boot 2.7
MyBatis
JDK 1.8

数据库

MySQL 8.0+
Maven
HikariCP

前端技术

HTML5 + CSS3
jQuery + BS3
响应式设计

安全 & 工具

登录拦截器
AOP 日志
Git 版本控制

核心功能

  • 用户注册 / 登录（MD5 加密）
  • 个人资料修改 & 头像上传
  • 收货地址管理（增删改查 + 默认地址）
  • 省市区三级联动
  • 商品热销排行（首页推荐）
  • 购物车（加入/删除/数量修改）
  • 订单生成 & 支付模拟
  • 订单列表 & 订单详情

注：本项目为课程实训作品，仅供学习交流使用。
